Setup guide — how to get these credentials

Email (takes ~2 minutes)

  1. Go to myaccount.google.com → Security → 2-Step Verification → enable it
  2. Same page → App passwords → name it "LTQ" → Generate
  3. Copy the 16-character password → paste into App password above
  4. Gmail address = your Google account email. From = any display name you want.

WhatsApp (takes a few hours for template approval)

  1. Go to developers.facebook.com → My Apps → Create App → Business
  2. Add WhatsApp product to your app
  3. App Dashboard → WhatsApp → API Setup → copy Phone Number ID
  4. Same page → generate a Permanent Token (System User in Meta Business Suite)
  5. Meta Business Suite → WhatsApp → Message Templates → Create:
    • Name: ltq_notification, Category: UTILITY, Language: English
    • Body: Hello {{1}},

      {{2}}

      — Liberia Talent Quest
  6. Submit → wait for approval (UTILITY templates usually approved within a few hours)

Both channels work independently — if WhatsApp isn't configured, emails still go through.
