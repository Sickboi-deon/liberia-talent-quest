Notification Channels

Configure email and WhatsApp delivery for contestant notifications.

Email (Gmail SMTP)
checking…
Generate at myaccount.google.com → Security → App passwords
Format: Display Name <email@gmail.com>
WhatsApp (Meta Cloud API)
checking…
From Meta App Dashboard → WhatsApp → API Setup
Permanent Token from Meta Business Suite
Phone number must have WhatsApp installed
Setup guide — how to get these credentials

Email (takes ~2 minutes)

  1. Go to myaccount.google.com → Security → 2-Step Verification → enable it
  2. Same page → App passwords → name it "LTQ" → Generate
  3. Copy the 16-character password → paste into App password above

WhatsApp (takes a few hours for template approval)

  1. developers.facebook.com → My Apps → Create App → Business → add WhatsApp product
  2. App Dashboard → WhatsApp → API Setup → copy Phone Number ID + generate Permanent Token
  3. Meta Business Suite → WhatsApp → Message Templates → Create (Name: ltq_notification, Category: UTILITY, Language: English, Body: Hello {{1}}, {{2}} — Liberia Talent Quest)
  4. Submit → wait for approval

Both channels work independently.
